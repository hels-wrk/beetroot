1) What is branch hell and when it appear?
  Это когда уходит больше времени на исправление конфликтов при слиянии ветки с основной веткой, чем ее разработка.
  Это происходит потому, что разработчик долго не выполняет обратного слияния, и основная ветка росла, если над проэктом работает не один разработчик.
2) What is reverse proxy server?
  Это сервер, который находится перед веб-серверами и пересылает запросы клиентов этим веб-серверам.
  Используеться для:
  -балансировка нагрузки
  -защиты от атак
  -кэширования
  -шифрование SSL
3) What is diff between git merge and git rebase?
  Обе команды предназначены для включения изменений из одной ветки в другую, но git merge создает новый комит включающими в себя все из обеих веток,
  существующие ветки никак не изменяются, в отличии от git rebase, которая перезаписывает историю проекта, создавая новые коммиты для каждого коммита в исходной ветке.
